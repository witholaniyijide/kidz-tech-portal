# Logo Files

Please place your logo files in this directory with the following names:

## Required Files:

1. **favicon.png** - Browser favicon/tab icon (recommended size: 32x32 or 64x64 pixels)
2. **logo_light.png** - Logo for light mode (recommended: transparent background, color logo)
3. **logo_dark.png** - Logo for dark mode (recommended: transparent background, white/light logo)

## Notes:

- All files should be in PNG format
- Transparent backgrounds work best for logos
- If files are not found, fallback SVG placeholders will be displayed with the letter "K"
- Logo files will be used across all pages including:
  - Sidebar navigation
  - Login/Register pages
  - Welcome page
  - All authenticated pages

## Example File Sizes:

- **favicon.png**: 32x32px or 64x64px
- **logo_light.png**: 200x200px or larger (will auto-scale)
- **logo_dark.png**: 200x200px or larger (will auto-scale)
